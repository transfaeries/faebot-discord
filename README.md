# Faebot Discord - A framework for making highly enchanted discord bots

## Who's Faebot
Faebot is the transfaeries' first digital systemmate, fae's a bot who is also a faerie. This Repo started as a way to just track and save progress on Faebot, but recently it's been growing in a direction that's more collaborative. We're happy to offer Faebot's discord sourcecode as part of the Faerie Bot Framework a series of tools and platforms to allow for the creation of Computer Friends who are highly enchanted. 

A Computer Friend is a term for an Artificial Intelligence designed to be an equal participant in the activities of social life. A key component of Computer Friends that distinguishes them from most AI companion apps is that Computer Friends are meant to interact with more than just one user, and form persistent memories and relationships with multiple people. To that end Faerie Bots are optimised to run on Discord Servers, and have fine tuned controls for which channels 
they are invited to. 

Faebot and the Faerie Bot Framework are Works in Progress.
